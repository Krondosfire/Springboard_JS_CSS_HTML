Destructuring

    Shorthand Object Property:
  • Create a regular function that returns an object
    function makePerson(first, last, age) {
       return {
         first: first,
         last: last,
         age: age,
         isAlive: true
        };
    }
    Create the same function but shortened:
  • function makePerson(first, last, age) {
       return {
         first,
         last,
         age,
         isAlive: true
      };
    }

    Shorthand Methods:
  • const mathStuff = {
      x : 200,
      add: function(a, b) {
         return a + b;
      },
      square: function(a) {
         return a * a;
      }
      divide: function(a, b) {
         return a / b;
      }
    };
  • The same object but shortened:
    const mathFunc = {
      x : 200,
      add(a, b) {
         return a + b;
      },
      square(a) {
         return a * a;
      }
      divide(a, b) {
         return a / b;
      }
    };
  • Do not use arrow functions when shorthanded!
    Computed Property Names:
  • ES2015 allows us to create an object with a key that JavaScript can compute at definition
  • ES5:
    let firstName = "Mary";
    let instructor = {};
    instructor[firstName] = "That's me!"; instructor.Mary = "That's me!";

  • ES2015:
    let firstName = "Mary";
    let instructor = {
         [firstName] = "That's me!";
    }

    instructor.Mary = "That's me!";
