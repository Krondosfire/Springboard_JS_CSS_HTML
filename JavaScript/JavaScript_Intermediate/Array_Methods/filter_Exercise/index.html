Array Methods: filter() Exercise

In this exercise, you will practice using the array method filter().

You are provided with an array of planet objects. Each object has three properties:name, temperature (K), and distance from the Sun (in astronomical units - AU).

    The task is to filter out planets based on the following criteria for potential habitability:
  • The temperature is between 253 K and 323 K, inclusive.
  • The distance from the Sun is between 0.75 and 1.5 AU, inclusive

Print the new array containing the names of the planets that match the criteria.

Step-by-step

filter() Method

  • Filter habitable planets based on name, temperature, and distance
  • Print the new array

Append the output to the HTML element

  • The resulting array contains the habitable planets with some details like Temperature, and distance
