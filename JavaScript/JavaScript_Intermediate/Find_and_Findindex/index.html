Array Methods

find and findIndex

Open the console in your browser to see the output of the JavaScript code.

    Properties of find() method:
  • Iterates through an array
  • Runs a callback function on each value in the array
  • If the callback returns true at any point, return the value in the array that we're iterating over
  • Otherwise, return undefined
    Properties of findIndex() method:
  • Iterates through an array
  • Runs a callback function on each value in the array
  • If the callback returns true for any single value, return the index at which that value is found.
    Own find() method:
  • Iterates through an array
  • Runs a callback on each value in the array
  • If the callback returns true at any point, return the value we're iterating over
  • Otherwise, return undefined
    Own findIndex() method:
  • Iterates through an array
  • Runs a callback on each value in the array
  • If the callback returns true for any single value, return the index at which that value is found
  • Otherwise, returns -1
    When You Would Use map() method:
  • Wnat to "transform" an array into another array of the same length
  • Do not want to overwrite an existing array and instead return a new copy
    filter() method:
  • Creates a new array
  • Loops through an array
  • Runs a callback function on each value in the array
  • If the callback function returns true, that value is pushed to the new array
  • If the callback function returns false, that value will not be included in the new array
  • The result of the callback will always be evaluated into a boolean
